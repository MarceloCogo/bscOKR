# Organizacao

A estrutura organizacional define contexto, visibilidade e responsabilidade dentro da plataforma.

## Estrutura

- A empresa pode ter niveis hierarquicos (ex.: empresa, diretoria, area, time).
- Cada no organizacional pode ter lider e membros.
- O contexto ativo define o recorte de dados exibido ao usuario.

## Contexto ativo

- O usuario escolhe contexto no topo da aplicacao.
- O contexto impacta mapa, objetivos, KRs e dashboards.
- Se o contexto salvo ficar invalido, o sistema aplica fallback automaticamente.

## Membresia e lideranca

- Membro: visualiza/atua conforme permissao.
- Lider do no: pode editar no proprio escopo quando permitido.

## Boas praticas

- Manter arvore organizacional atualizada.
- Evitar duplicidade de nos com mesma funcao.
- Definir responsavel por no para acelerar governanca.
