# Key Results (KRs)

KRs transformam objetivos em indicadores mensuraveis.

## Tipos de KR

- **AUMENTO**: quanto maior, melhor.
- **REDUCAO**: quanto menor, melhor.
- **LIMIAR**: precisa ficar acima/abaixo de limite alvo.
- **ENTREGAVEL**: acompanhamento por checklist de entregas.

## Campos importantes

- Titulo
- Tipo
- Valor atual
- Valor alvo
- Unidade
- Ciclo
- Status

## Regras de negocio

- Todo KR deve estar ligado a um objetivo.
- O progresso e calculado conforme tipo e valores.
- KR de tipo ENTREGAVEL usa checklist para calcular progresso.
- Historico de atualizacao e registrado para auditoria e acompanhamento.

## Boas praticas

- Definir target objetivo e verificavel.
- Evitar KR ambiguo sem formula de medicao.
- Atualizar periodicamente para manter previsibilidade.
- Usar entregavel apenas para marcos concretos.

## Erros comuns

- KR sem unidade clara.
- KR sem frequencia de atualizacao definida.
- Misturar varias metas diferentes em um unico KR.
