# Objetivos Estrategicos

Os objetivos estrategicos representam os resultados de negocio que orientam a execucao.

## Campos principais

- Titulo
- Descricao (opcional)
- Regiao do mapa
- Perspectiva
- Pilar (quando aplicavel)
- Status
- Sponsor (responsavel)

## Regras de negocio

- Objetivos pertencem a um contexto organizacional.
- O usuario so pode criar/editar objetivo se tiver permissao no contexto.
- Objetivos da regiao **AMBITION** seguem regras especiais:
  - existe apenas um por contexto,
  - perspectiva e sempre **Ambicao Estrategica**,
  - pilar nao se aplica.

## Criacao

1. Acesse o mapa estrategico.
2. Selecione a regiao.
3. Informe titulo e campos obrigatorios.
4. Salve e valide se o objetivo apareceu no painel.

## Edicao

- Objetivos podem ter titulo, descricao, status e sponsor alterados.
- Para objetivo de Ambicao, perspectiva e pilar ficam bloqueados.

## Exclusao

- Deve ser usada com cuidado.
- Ao excluir objetivo, KRs relacionados deixam de estar vinculados ao mapa (avaliar impacto antes).
