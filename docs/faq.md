# FAQ

## 1) Como entrar no sistema?

- Priorize **Continuar com Microsoft** quando sua organizacao usa Entra ID.
- Para login local, use slug + email + senha.

## 2) Nao consigo ver dados de outra area. E erro?

Nao necessariamente. O sistema aplica escopo organizacional por contexto e permissao.

## 3) Posso ter mais de uma Ambicao Estrategica?

Nao. Existe uma Ambicao por contexto, com perspectiva fixa e sem pilar.

## 4) Como acompanhar se o SCIM esta funcionando?

Em **Admin > Configuracao**, use o bloco de monitoramento SCIM para ver sucessos, falhas e eventos recentes.

## 5) O que fazer se um KR estiver sem progresso?

- Verifique se o KR recebeu atualizacoes no ciclo atual.
- Confirme se os valores e unidade estao corretos.
- Para KR entregavel, revise checklist e itens concluidos.

## 6) Quem pode editar configuracoes globais?

Usuarios com permissao de gerenciamento de configuracao (tipicamente administradores).
